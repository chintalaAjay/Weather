# Weather 🌦️

A simple and responsive Weather Application that displays real-time weather information using the OpenWeatherMap API.

## 🔗 Live Demo
👉 https://weather-5f1c9.web.app

---

## 📌 Features

- 🌍 Search weather by city name
- 🌡️ Displays real-time temperature and weather conditions
- 🎨 Clean and responsive user interface
- ☁️ Dynamic weather background images
- 🚀 Hosted using Firebase Hosting

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Firebase Hosting
- OpenWeatherMap API

---

---

## ⚙️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/chintalaAjay/Weather.git
```

2. Open the project folder

```bash
cd Weather
```

3. Open the project in VS Code

4. Install the **Live Server** extension in VS Code

5. Right-click on `home.html`

6. Click **Open with Live Server**
